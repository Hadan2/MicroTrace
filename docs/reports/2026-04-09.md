# 04.09 트러블슈팅

## 문제 1: WSL2에서 프론트엔드 접속 불가
- **증상:** `http://172.17.238.189:5173` 접속 안 됨
- **원인:** Vite 기본 설정이 `127.0.0.1`에만 바인딩 → WSL2 외부에서 접근 불가
- **해결:** `vite.config.ts`에 `server: { host: true }` 추가 → `0.0.0.0` 바인딩

## 문제 2: UI 실시간 업데이트 안 됨 (새로고침해야 바뀜)
- **증상:** WebSocket 메시지는 오는데 화면이 안 바뀜
- **원인:** React `StrictMode`가 개발 모드에서 `useEffect`를 두 번 실행(mount → unmount → mount) → 첫 번째 WebSocket이 cleanup되면서 두 번째 연결 불안정
- **해결:** `main.tsx`에서 `StrictMode` 제거

## 문제 3: p50/p99 값이 안 바뀌는 것처럼 보임
- **증상:** 화면 값이 고정된 것처럼 보임
- **원인:** testenv(docker-compose)를 실행하지 않은 상태였음. WSL2 일반 트래픽(Google, Microsoft, Anthropic)은 RTT가 거의 일정해서 p99가 수렴
- **해결:** testenv 실행 후 확인 → 정상 동작 확인
